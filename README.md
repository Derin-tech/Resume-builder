# Next-Gen Resume Builder ✨

A modern, dynamic, AI-powered Resume Builder application crafted with React, Vite, Tailwind CSS, Zustand, and Firebase.

## ⚡ Core Features
- **Intelligent Cloud Persistence**: Debounced real-time data synchronizations managed by Firebase Firestore with secure Email + Google Auth.
- **AI Career Assistant**: Powered by Anthropic's Claude 3.5 Sonnet to restructure and optimize your writing impact via direct semantic matching.
- **Native ATS PDF Export**: Fully vector-rendered text generation powered by `@react-pdf/renderer` ensuring Applicant Tracking Systems read your files correctly.
- **Global Application Store**: Built with Zustand mapping instant keyboard undo/redo history matrices across the entire UI.
- **Responsive Dynamic Themes**: Switch instantly between Classic, Modern, and Executive stylistic templates while editing across multiple breakpoint environments!

## 💻 Local Development

1. **Install Dependencies**
   ```bash
   npm install
   ```
2. **Setup Environment Variables**
   Create a `.env.local` file mapping your Firebase architecture properties.
3. **Launch Server**
   ```bash
   npm run dev
   ```
